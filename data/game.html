Pilih perintah yang lebih spesifik

Perintah Deskripsi
$game --populasi : Meningkatkan populasi
$game --tippm : Tips untuk pemula
$game --hirarki : Tutorial hirarki jalan (Agar tidak macet)
$game --lag : Game ngelag? Ini ketik perintah untuk solusi
$game --dsa : Tutorial DSA
$game --redeem : Cara redeem code
$game --budaya : Bangunan dengan budaya
$game --pelabuhan : Cara membuat zona pelabuhan
$game --ro-memory : Error: Run out Memory? Ketik peritah untuk solusi!
$game --apk : Download apk theotown

Ketik ulang peritah sesuai petunjuk diatas.
Contoh $game --dsa
